# mouse-pan
Smooth paning on mouse move

example
===
https://codepen.io/jackismissing/pen/prEjpx

usage
====

Import
---
```
import MousePan from 'mouse-pan';
```

Instantiation
---
```
const mousePan = new MousePan({
    el: document.querySelector('.pan-wrapper'),
    ease: .08
});
```

Options
---
ease: default to 0.8;
